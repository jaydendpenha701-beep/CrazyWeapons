# CrazyWeapons Plugin

A powerful Minecraft plugin for Paper 1.21.1 that adds unique custom weapons.

## 🚀 Features
- **Lightning Sword**: Strike visual lightning and deal +4 damage on hit.
- **Fire Wand**: Shoot small fireballs (no block damage).
- **Lifesteal Blade**: Heal 2 hearts on every hit.

## 🛠️ Crafting Recipes
### 1. Lightning Sword
- [Copper] [Diamond] [Copper]
- [Diamond] [**Diamond Sword**] [Diamond]
- [Copper] [Diamond] [Copper]

### 2. Fire Wand
- [Magma Cream] [Fire Charge] [Magma Cream]
- [Fire Charge] [**Blaze Rod**] [Fire Charge]
- [Magma Cream] [Fire Charge] [Magma Cream]

### 3. Lifesteal Blade
- [Fermented Eye] [Ghast Tear] [Fermented Eye]
- [Ghast Tear] [**Netherite Sword**] [Ghast Tear]
- [Fermented Eye] [Ghast Tear] [Fermented Eye]

## 📦 Files
- **Plugin JAR**: `CrazyWeapons.jar` (Located in the root directory)
- **Resource Pack**: `CrazyWeaponsPack/` (Download this folder for custom textures)
- **Source Code**: `src/main/java/com/github/crazyweapons/CrazyWeapons.java`

## 🏗️ Building from Source
If you want to build the plugin yourself, ensure you have **Java 21** installed and run:
```bash
mvn clean package
```
The resulting JAR will be in the `target/` directory.
